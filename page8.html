IT Rendszermérnök / Linux

Megbízóm részére a XIII. kerületbe, hybrid munkavégzéssel keresek IT Rendszermérnököt (Linux)

Feladatai:
- a cég fizikai és virtuális kiszolgálóin (Hyper-V), Failover Cluster-ek üzemeltetése,
karbantartása, szoftverek telepítése
- elsősorban RHEL Linux platformon kialakított My SQL/Oracle adatbázis és Apache/Nginx alkalmazás szerverek produktív teszt környezetének üzemeltetése.
- Midlewear alkalmazások (WSO2 API-Gateway; Artemis MQ message broker) üzemeltetése
- alkalmazás szintű beállítások, hibafeltárás/elhárítás, teljesítményoptimalizálás végrehajtása
- Alkalmazás verziók kezelése, (telepítés, konfigurálás, verzió frissítés)
- Hardening/sebezhetőségi riportok alapján a konfigurációk frissítése, nyilvántartások vezetése
- Fejlesztési terület szakértői támogatása, dokumentációk készítése

Elvárások:

- Linux (Red Hat) adminisztrátori szintű ismerete
- Monitoring rendszerek (Nagios/Munin) ismerete
- Docker virtualizációs technológia ismerete
- Shell script nyelvek (Bash, perl, Python) ismerete
- Alapvető infrastruktúra ismeretek (hálózat, tűzfal, loadbalancer, DNS, syslog stb.)
- Alapszintű adatbázis( MySQL, Oracle,, MariaDB, MongoDB) ismeretek
- Webszerverek (Nginx, Apache, HAProxy) ismerete
- Verziókövető rendszerek (SVN, Git, GitLab) ismerete
- minimum középfokú informatikai végzettség
- minimum 3 év szakmai tapasztalat hasonló munkakörben
- angol nyelv középfokú ismerete, szakmai angol nyelv ismerete
